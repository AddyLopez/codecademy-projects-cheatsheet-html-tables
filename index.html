HTML Table Reference

Table Tags

Tag Name Description Use
<table> Table The wrapper element for all HTML tables Signals a table
<thead> Table Head The set of rows defining the column headers in a table Lends semantic meaning to the row of column headers
<tbody> Table Body The set of rows containing actual table data Lends semantic meaning to the set of rows containing data
<tr> Table Row The table row container (use: creates a new row) Signals a new row
<th> Table Heading The table heading container Labels and clarifies the meaning of data
<td> Table Data The container for a datum in a table Displays a piece of information
<tfoot> Table Foot The set of rows defining the footer in a table Typically displays sums, differences, results of tabular data, etc.

Table Attributes

Attribute Name Use Value(s)
scope Scope Specifies that a <th> cell is a header for a column, a row, or a group of either. "col," "row," "colgroup," or "rowgroup"
colspan Column Span Defines how many columns a <td> should span. An integer greater than or equal to 1.
rowspan Row Span Defines how many rows a <td> should span. An integer greater than or equal to 1
